# ☁️ CloudStream için Türkçe Eklentiler

## 💾 Kurulum

1. **[cloudstream/releases](<a href="https://github.com/recloudstream/cloudstream/releases" target="_blank" rel="noopener noreferrer"> indir </a>)** _Adresinden güncel APK dosyasını indirip kurun._
2. Eğer bazı servisler düzgün çalışmıyorsa. **[cloudstream/prereleases](<a href="https://github.com/recloudstream/cloudstream/releases/download/pre-release/app-prerelease-release.apk" target="_blank" rel="noopener noreferrer"> indir </a>)** _Adresinden en son çıkan APK dosyasını indirip kurun._
3. **Uygulamayı açın `Ayarlar` - `Eklentiler` - `Depo Ekle` kısmına girin.**
4. **Sağdaki Depo Url'sini kopyalayıp yapıştırın.** `https://shorturl.at/mvP78` or `https://raw.githubusercontent.com/testermania/tester-cloudstream/master/repo.json`
